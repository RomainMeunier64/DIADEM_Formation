## 📚 Programme détaillé

### 00 — Introduction
- Vérification de l'environnement et des librairies
- Rappels Python : types, listes, dictionnaires, fonctions
- Introduction à NumPy : tableaux, opérations vectorisées

### 01 — Pandas & Manipulation de données
- Création et chargement de DataFrames
- Exploration, sélection et filtrage
- Nettoyage des données (valeurs manquantes, doublons)
- Transformation, nouvelles colonnes, agrégations avec `groupby()`

### 02 — Visualisation
- **Matplotlib** : graphiques de base, subplots, personnalisation
- **Seaborn** : distributions, boxplots, heatmaps, pairplot
- **Plotly** : graphiques interactifs pour la présentation
- Guide : quel graphique choisir selon vos données

### 03 — Statistiques
- Mesures de tendance centrale et de dispersion
- Distributions (normale, uniforme, asymétrique)
- Corrélations et coefficient de Pearson
- Tests statistiques : test t, Chi-2, Shapiro-Wilk, p-value
- Théorème Central Limite

### 04 — Machine Learning
- Pipeline ML standard (préparation → entraînement → évaluation)
- Classification : comparaison de modèles (Random Forest, SVM, KNN...)
- Régression : prédiction de valeurs continues
- Validation croisée et optimisation des hyperparamètres
- Overfitting vs underfitting
